## 学习路径

- **路径名称**：旋度与散度核心概念构建与应用路径
- **总阶段数**：4

### 阶段 1：夯实数学基础，建立旋度与散度的核心概念认知，理解其物理意义
- **行动项**：回顾矢量分析中的关键数学工具（偏导数、线面积分、矢量运算规则）；研读旋度与散度的定义、计算公式及几何/物理含义（旋度对应旋转源，散度对应发散源）；结合电磁学场景，初步关联旋度散度与电磁场的源特性（如磁场的旋度、电场的散度）
- **推荐资源**：课程讲解文档；视频学习资料；知识点思维导图(Mermaid)
- **检查点**：能独立复述旋度与散度的定义、计算公式，准确说出两者对应的物理意义，完成3道基础概念辨析题（正确率≥80%）
### 阶段 2：掌握旋度与散度的计算方法，熟练完成基础题型的推导与运算
- **行动项**：针对笛卡尔坐标系、柱坐标系、球坐标系下的旋度散度计算规则进行专项练习；完成不同矢量场的旋度散度计算习题，重点攻克含坐标系转换的复杂题型；对照习题解析，梳理计算过程中的易错点（如偏导顺序、坐标系参数差异）
- **推荐资源**：分层练习题(含答案与解析)；课程讲解文档
- **检查点**：能独立完成5种典型矢量场的旋度散度计算，涵盖不同坐标系，计算正确率≥85%，能清晰指出计算过程中的关键易错点
### 阶段 3：关联电磁学核心定理，理解旋度散度在电磁场理论中的应用逻辑
- **行动项**：推导麦克斯韦方程组的微分形式，明确旋度散度在方程中的具体体现；分析电磁场源与场的关系（如电荷对应电场散度，电流对应磁场旋度）；研读拓展阅读材料，梳理旋度散度在电磁波传播、电磁感应等场景的应用案例
- **推荐资源**：拓展阅读材料；课程讲解文档；视频学习资料
- **检查点**：能完整推导麦克斯韦方程组的微分形式，准确说明每个方程中旋度或散度的物理意义，能结合1-2个电磁学案例阐述旋度散度的应用逻辑
### 阶段 4：综合应用知识解决复杂问题，适配课程考核要求，实现知识体系闭环
- **行动项**：完成综合练习题，涵盖旋度散度计算、定理推导、电磁学综合应用等题型；整理知识体系，绘制包含数学基础、核心概念、电磁学应用的知识思维导图；针对课程考核高频考点进行专项复盘，查漏补缺
- **推荐资源**：分层练习题(含答案与解析)；知识点思维导图(Mermaid)；课程讲解文档
- **检查点**：完成10道综合应用题（涵盖计算、推导、案例分析），正确率≥90%，能独立绘制完整的知识思维导图，清晰呈现旋度散度与电磁学知识体系的关联

### 推送策略
- **日常推送规则**：每日推送1条核心概念卡片（含定义、公式、1道基础例题）；每日推送1道专项计算练习题，聚焦当日学习的核心计算方法
- **自适应规则**：若基础概念题正确率＜80%，自动追加1条概念解析视频链接+1组对比辨析练习题；若综合应用题正确率＜85%，自动推送对应知识点的拓展阅读材料+高频错题专项解析；若连续3日完成推送任务且正确率≥90%，调整推送内容为综合题型，强化知识整合能力
