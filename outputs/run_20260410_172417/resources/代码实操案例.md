### 实操步骤（分步拆解，代码可直接复制运行）
#### 步骤1：准备模拟财务数据集（贴合会计场景）
无需额外找数据，直接用代码生成包含会计核心指标的模拟数据集，包含“资产负债率、净利润率、流动比率”3个财务指标，以及标签“财务风险（0=无风险，1=有风险）”，数据逻辑贴合会计专业认知。
```python
import pandas as pd
import numpy as np

# 设置随机种子，保证结果可复现（新手友好）
np.random.seed(42)

# 生成100家企业的财务指标数据（模拟会计常见指标）
data = pd.DataFrame({
    # 资产负债率：会计核心指标，反映偿债能力
    "资产负债率": np.random.uniform(30, 90, 100),
    # 净利润率：反映盈利能力，会计核心关注指标
    "净利润率": np.random.uniform(-5, 25, 100),
    # 流动比率：反映短期偿债能力，会计常用指标
    "流动比率": np.random.uniform(0.8, 3.5, 100),
    # 财务风险标签：0=无风险，1=有风险（基于会计指标阈值生成）
    "财务风险": [1 if (负债 > 70 or 利润率 < 0 or 比率 < 1.2) else 0 
                for 负债, 利润率, 比率 in zip(data["资产负债率"], data["净利润率"], data["流动比率"])]
})

# 查看前5行数据，熟悉结构（会计指标一目了然）
print("模拟财务数据集（前5行）：")
print(data.head())
```

#### 步骤2：数据预处理（简单易懂，贴合入门需求）
机械学习的核心前提之一是“数据干净”，此处仅需做最基础的**特征与标签拆分**，无复杂清洗逻辑，适配入门阶段，避免信息过载。
```python
# 拆分特征（X）和标签（y）
# X：用于预测的财务指标（会计核心数据）
X = data[["资产负债率", "净利润率", "流动比率"]]
# y：需要预测的目标（财务风险标签）
y = data["财务风险"]

print("\n特征数据（前5行）：")
print(X.head())
print("\n标签数据（前5行）：")
print(y.head())
```

#### 步骤3：构建决策树模型（机械学习核心实操）
选择**决策树模型**作为入门首选——逻辑直观（像会计做决策时的判断流程，比如“资产负债率过高→判定风险”），无需复杂调参，10行代码即可完成建模，完美匹配稳健型学习节奏。
```python
from sklearn.tree import DecisionTreeClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score, classification_report

# 1. 划分训练集和测试集（70%训练，30%测试，符合入门项目常规比例）
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 2. 初始化决策树模型（默认参数，入门无需调参，降低难度）
model = DecisionTreeClassifier()

# 3. 训练模型（让模型学习财务指标与风险的关联规律）
model.fit(X_train, y_train)

# 4. 用测试集预测（验证模型效果）
y_pred = model.predict(X_test)

# 5. 评估模型（核心指标，直观易懂）
accuracy = accuracy_score(y_test, y_pred)
print(f"\n模型准确率：{accuracy:.2f}（越高说明财务风险预测越准）")
print("\n分类报告（细节指标，可选看）：")
print(classification_report(y_test, y_pred, zero_division=0))
```

#### 步骤4：可视化决策树（直观理解模型逻辑，贴合会计决策思维）
决策树的优势是“可解释性强”，可视化后能看到模型如何基于会计指标做风险判断，类似会计做决策的逻辑链条，帮助入门者理解机械学习模型的决策过程，而非仅停留在代码层面。
```python
from sklearn.tree import plot_tree
import matplotlib.pyplot as plt

# 设置图表中文显示（解决乱码问题）
plt.rcParams["font.sans-serif"] = ["SimHei"]
plt.rcParams["axes.unicode_minus"] = False

# 绘制决策树
plt.figure(figsize=(12, 8))
plot_tree(
    model,
    feature_names=["资产负债率", "净利润率", "流动比率"],  # 会计核心指标名称
    class_names=["无风险", "有风险"],  # 风险标签，贴合会计场景
    filled=True,  # 用颜色区分风险类别，直观易懂
    rounded=True  # 节点边框圆角，提升可读性
)
plt.title("财务风险预测决策树（会计指标驱动）", fontsize=14)
plt.show()
```
