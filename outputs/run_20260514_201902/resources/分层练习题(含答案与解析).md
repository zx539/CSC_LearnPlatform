## 第一层：基础巩固层（目标：夯实核心计算，扫清基础易错点）
建议完成时间：45分钟
### 题1
【题干】求信号 $f(t) = e^{-2t}\cos(3t)u(t) + t e^{-t}u(t-1)$ 的单边拉氏变换。
【答案】
$$F(s) = \frac{s+2}{(s+2)^2 + 9} + \frac{(s+2)e^{-(s+1)}}{(s+1)^2} \quad (\text{Re}[s]>-1)$$
【解析】
考点为拉氏变换的**频移性质、时移性质**：
1. 第一部分 $e^{-2t}\cos(3t)u(t)$ 直接用频移性质：已知 $\mathcal{L}[\cos3t u(t)] = \frac{s}{s^2+9}$，频移2个单位得 $\frac{s+2}{(s+2)^2 + 9}$，收敛域 $\text{Re}[s]>-2$。
2. 第二部分需配凑时移标准形式 $f(t-t_0)u(t-t_0)$：
   $$t e^{-t}u(t-1) = e^{-1}\cdot\left[(t-1)e^{-(t-1)} + e^{-(t-1)}\right]u(t-1)$$
   已知 $\mathcal{L}[t e^{-t}u(t)] = \frac{1}{(s+1)^2}$，$\mathcal{L}[e^{-t}u(t)] = \frac{1}{s+1}$，用时移性质得变换结果为 $e^{-(s+1)}\left(\frac{1}{(s+1)^2} + \frac{1}{s+1}\right)$，收敛域 $\text{Re}[s]>-1$。
3. 最终收敛域取两部分交集 $\text{Re}[s]>-1$。

### 题2
【题干】已知信号的单边拉氏变换为 $F(s) = \frac{s+3}{s^2+3s+2}$，求信号的初值 $f(0^+)$ 和终值 $f(\infty)$。
【答案】$f(0^+)=1$，$f(\infty)=0$
【解析】
考点为**初终值定理的适用条件与计算**：
1. 初值定理适用条件：$F(s)$ 为真分式（分子阶数<分母阶数），本题满足。初值计算：
   $$f(0^+) = \lim_{s \to \infty} sF(s) = \lim_{s \to \infty} \frac{s^2+3s}{s^2+3s+2} = 1$$
2. 终值定理适用条件：$F(s)$ 的所有极点都在s左半平面，或仅在s=0处有单极点。本题极点为 $s=-1, s=-2$，均在左半平面，满足条件。终值计算：
   $$f(\infty) = \lim_{s \to 0} sF(s) = \lim_{s \to 0} \frac{s(s+3)}{s^2+3s+2} = 0$$

### 题3
【题干】已知单边拉氏变换 $F(s) = \frac{2s+5}{s^2+3s+2}$，收敛域 $\text{Re}[s]>-1$，求逆变换 $f(t)$。
【答案】$f(t) = (3e^{-t} - e^{-2t})u(t)$
【解析】
考点为**部分分式展开法求逆变换**：
1. 对 $F(s)$ 做部分分式分解：
   $$F(s) = \frac{2s+5}{(s+1)(s+2)} = \frac{A}{s+1} + \frac{B}{s+2}$$
   代入极点计算得 $A=3, B=-1$，即 $F(s) = \frac{3}{s+1} - \frac{1}{s+2}$。
2. 收敛域 $\text{Re}[s]>-1$ 在两个极点的右侧，对应因果信号，逆变换得上述结果。

## 第二层：强化提升层（目标：突破薄弱考点，掌握综合解题逻辑）
建议完成时间：1.5小时
### 题1
【题干】求信号 $f(t) = e^{-t}u(t) + e^{2t}u(-t)$ 的双边拉氏变换与收敛域；若已知某信号的双边拉氏变换与上述结果相同，分别求收敛域为 $\text{Re}[s]>2$ 和 $\text{Re}[s]<-1$ 时对应的逆变换。
【答案】
1. 所求双边拉氏变换为 $F(s) = \frac{-3}{(s+1)(s-2)}$，收敛域 $-1<\text{Re}[s]<2$；
2. 收敛域 $\text{Re}[s]>2$ 时，逆变换为 $f(t) = (e^{-t} - e^{2t})u(t)$；
3. 收敛域 $\text{Re}[s]<-1$ 时，逆变换为 $f(t) = (-e^{-t} + e^{2t})u(-t)$。
【解析】
考点为**双边拉氏变换的收敛域物理意义**：
- 因果信号 $e^{-t}u(t)$ 的双边拉氏变换为 $\frac{1}{s+1}$，收敛域 $\text{Re}[s]>-1$；反因果信号 $e^{2t}u(-t)$ 的双边拉氏变换为 $\frac{-1}{s-2}$，收敛域 $\text{Re}[s]<2$，两者交集为 $-1<\text{Re}[s]<2$，得到总变换。
- 收敛域在最右侧极点（s=2）右侧时，所有分式对应因果信号；收敛域在最左侧极点（s=-1）左侧时，所有分式对应反因果信号。

### 题2
【题干】RC串联电路中 $R=1\Omega, C=1F$，输入电压 $e(t) = e^{-2t}u(t)$，初始电容电压 $u_C(0^-)=1V$，用复频域分析法求零输入响应、零状态响应、全响应的时域表达式。
【答案】
- 零输入响应：$u_{Czi}(t) = e^{-t}u(t)$
- 零状态响应：$u_{Czs}(t) = (e^{-t} - e^{-2t})u(t)$
- 全响应：$u_C(t) = (2e^{-t} - e^{-2t})u(t)$
【解析】
考点为**复频域电路建模与响应求解**：
1. 画出s域电路模型：电容等效为 $\frac{1}{sC} = \frac{1}{s}$ 串联电压源 $\frac{u_C(0^-)}{s} = \frac{1}{s}$。
2. 零输入响应：输入短路，仅保留初始条件源，列KVL得 $(R+\frac{1}{sC})I_{zi}(s) = \frac{1}{s}$，解得 $U_{Czi}(s) = I_{zi}(s)\cdot\frac{1}{s} + \frac{1}{s} = \frac{1}{s+1}$，逆变换得结果。
3. 零状态响应：初始条件源短路，输入 $E(s) = \frac{1}{s+2}$，列KVL得 $(R+\frac{1}{sC})I_{zs}(s) = E(s)$，解得 $U_{Czs}(s) = \frac{E(s)}{s+1}\cdot\frac{1}{s} = \frac{1}{(s+1)(s+2)} = \frac{1}{s+1} - \frac{1}{s+2}$，逆变换得结果。

### 题3
【题干】已知因果线性时不变系统的系统函数为 $H(s) = \frac{s+1}{s^3 + 3s^2 + 3s + 2 + K}$，求K满足什么条件时系统稳定。
【答案】$-2<K<7$
【解析】
考点为**因果系统稳定条件与罗斯判据应用**：
1. 因果系统稳定的充要条件是：所有极点均在s左半平面，即特征方程 $s^3 + 3s^2 + 3s + 2+K=0$ 的所有根实部为负。
2. 列罗斯阵列：
   $$
   \begin{align}
   s^3: &\quad 1 \quad 3 \\
   s^2: &\quad 3 \quad 2+K \\
   s^1: &\quad \frac{3\times3 - 1\times(2+K)}{3} = \frac{7-K}{3} \\
   s^0: &\quad 2+K
   \end{align}
   $$
3. 罗斯阵列第一列全部为正时系统稳定，即 $\frac{7-K}{3}>0$ 且 $2+K>0$，解得 $-2<K<7$。

## 第三层：真题冲刺层（目标：熟悉真题套路，提升应试解题速度）
建议完成时间：1小时
### 题1（2022年西安电子科技大学考研真题原型）
【题干】已知因果线性时不变系统的微分方程为 $y''(t) + 3y'(t) + 2y(t) = f'(t) + 3f(t)$，输入 $f(t)=e^{-3t}u(t)$，初始条件 $y(0^-)=1, y'(0^-)=2$，求：
（1）系统函数 $H(s)$ 和单位冲激响应 $h(t)$；
（2）零输入响应 $y_{zi}(t)$、零状态响应 $y_{zs}(t)$、全响应 $y(t)$。
【答案】
（1）$H(s) = \frac{s+3}{s^2+3s+2} \ (\text{Re}[s]>-1)$，$h(t) = (2e^{-t} - e^{-2t})u(t)$
（2）$y_{zi}(t) = (4e^{-t} - 3e^{-2t})u(t)$，$y_{zs}(t) = (e^{-t} - e^{-2t})u(t)$，$y(t) = (5e^{-t} - 4e^{-2t})u(t)$
【解析】
考点为**微分方程的s域求解、系统函数与响应拆分**，为拉氏变换最高频考点：
1. 零状态下对微分方程两边做拉氏变换，得 $(s^2+3s+2)Y_{zs}(s) = (s+3)F(s)$，因此 $H(s) = \frac{Y_{zs}(s)}{F(s)} = \frac{s+3}{s^2+3s+2}$，部分分式分解后求逆变换得 $h(t)$。
2. 零输入响应：令输入 $F(s)=0$，拉氏变换时保留初始条件，得 $(s^2+3s+2)Y_{zi}(s) - s y(0^-) - y'(0^-) - 3y(0^-) = 0$，代入初始值求解后逆变换即可。
3. 零状态响应：$Y_{zs}(s) = H(s)F(s) = \frac{s+3}{(s^2+3s+2)(s+3)} = \frac{1}{s^2+3s+2}$，逆变换得结果，全响应为零输入+零状态响应之和。

### 题2（2023年电子科技大学考研真题原型）
【题干】已知线性时不变系统的双边拉氏变换为 $H(s) = \frac{s}{(s+1)(s-2)}$，收敛域为 $-1<\text{Re}[s]<2$，判断系统的因果性、稳定性，并求单位冲激响应 $h(t)$。
【答案】系统非因果、稳定，$h(t) = \frac{1}{3}e^{-t}u(t) - \frac{2}{3}e^{2t}u(-t)$
【解析】
考点为**双边拉氏变换下的因果性、稳定性判断**：
1. 因果性判断：因果系统的收敛域必须是最右侧极点的右半平面，本题收敛域在极点 $s=2$ 的左侧，因此是非因果系统。
2. 稳定性判断：系统稳定的充要条件是收敛域包含jω轴（$\text{Re}[s]=0$），本题收敛域 $-1<\text{Re}[s]<2$ 包含jω轴，因此稳定。
3. 逆变换求解：收敛域在 $s=-1$ 右侧，因此 $\frac{1}{s+1}$ 对应因果信号 $e^{-t}u(t)$；收敛域在 $s=2$ 左侧，因此 $\frac{1}{s-2}$ 对应反因果信号 $-e^{2t}u(-t)$，对 $H(s)$ 做部分分式分解后代入即可得到结果。

## 考点复盘总结（拉氏变换考研核心框架）
1. 基础计算类：单边拉氏变换/逆变换求解、常用变换对、核心性质（时移、频移、尺度、微分、积分）、初终值定理（适用条件为高频易错点）
2. 拓展核心类：双边拉氏变换求解、收敛域物理意义、不同收敛域对应的逆变换
3. 系统分析类：复频域电路/微分方程求解、零输入/零状态/全响应拆分、系统函数$H(s)$求解
4. 稳定性判断类：因果/非因果系统的稳定条件、罗斯判据应用、稳态响应求解
