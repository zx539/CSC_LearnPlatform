### Q1. 下列关于单边拉氏变换和双边拉氏变换的描述，错误的是？
- **题型**：single_choice
- **是否必填**：必填
- **评估维度**：阶段测试
- **可选项**：
  - A. 单边拉氏变换仅考虑信号t≥0的部分，双边拉氏变换考虑全时间域的信号
  - B. 同一个信号的单边拉氏变换和双边拉氏变换的收敛域一定相同
  - C. 因果信号的单边拉氏变换和双边拉氏变换表达式完全一致
  - D. 单边拉氏变换的微分性质需要考虑信号的初始条件，双边拉氏变换的微分性质不需要考虑初始值
### Q2. 已知信号f(t)=e^{-a|t|}，a>0，该信号的双边拉氏变换的收敛域为？
- **题型**：single_choice
- **是否必填**：必填
- **评估维度**：阶段测试
- **可选项**：
  - A. Re[s] > a
  - B. Re[s] < -a
  - C. -a < Re[s] < a
  - D. 不存在收敛域
### Q3. 下列关于拉氏变换收敛域的说法，错误的是？
- **题型**：single_choice
- **是否必填**：必填
- **评估维度**：阶段测试
- **可选项**：
  - A. 双边拉氏变换的收敛域是s平面上平行于虚轴的带状区域
  - B. 因果信号的单边拉氏变换收敛域一定位于其最右侧极点的右半平面
  - C. 拉氏变换的收敛域内不包含任何极点
  - D. 反因果信号的双边拉氏变换收敛域位于其最右侧极点的右半平面
### Q4. 已知信号f(t)的双边拉氏变换为$F_b(s)=\frac{1}{(s+1)(s-2)}$，若其收敛域为Re[s] < -1，则对应的时域信号为？

- **是否必填**：必填
- **评估维度**：阶段测试
- **可选项**：
  - A. $(e^{-t} - e^{2t})u(t)$
  - B. $(-e^{-t} + e^{2t})u(t)$
  - C. $-e^{-t}u(t) - e^{2t}u(-t)$
  - D. $e^{-t}u(-t) - e^{2t}u(-t)$
### Q5. 对于信号f(t)，其单边拉氏变换为$F_u(s)$，双边拉氏变换为$F_b(s)$，下列关于拉氏变换时移性质的说法正确的是？

- **是否必填**：必填
- **评估维度**：阶段测试
- **可选项**：
  - A. 若f(t)是任意信号，右移$t_0$后双边拉氏变换为$e^{-st_0}F_b(s)$
  - B. 若f(t)是任意信号，右移$t_0$后单边拉氏变换为$e^{-st_0}F_u(s)$
  - C. 若f(t)是反因果信号，右移$t_0$后单边拉氏变换等于原信号的单边拉氏变换
  - D. 双边拉氏变换的时移性质仅适用于因果信号
