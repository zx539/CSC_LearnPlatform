## 学习路径

- **路径名称**：考研信号与系统拉氏变换模块强化提分学习路径
- **总阶段数**：3

### 阶段 1：补全拉氏变换知识盲区，搭建覆盖全部考研考点的完整知识框架，攻克双边拉氏变换薄弱点
- **行动项**：1. 对照考点讲义梳理拉氏变换全模块考点，重点区分单边/双边拉氏变换的定义、收敛域、变换对、性质差异；2. 完成双边拉氏变换基础专项练习，整理错题对应的知识点盲区；3. 结合思维导图搭建个人专属拉氏变换知识框架，标注薄弱考点
- **推荐资源**：知识点思维导图(Mermaid)；考点梳理类讲义；视频学习资料（拉氏变换考点串讲部分）
- **检查点**：可独立默写拉氏变换完整考点框架，双边拉氏变换收敛域判断、正反变换计算类基础题正确率≥90%
### 阶段 2：突破复频域分析、系统稳定性判断核心高频考点，掌握各题型通用解题方法
- **行动项**：1. 逐点学习复频域系统函数、电路复频域建模、系统稳定性判据三类核心考点，结合真题拆解掌握解题步骤；2. 完成对应考点分层中档练习题，每类题型至少练15道，总结易错点；3. 整理拉氏变换常考题型清单，标注每类题型的解题思路
- **推荐资源**：真题拆解类讲解资源；分层练习题(含答案与解析)（中档题部分）
- **检查点**：复频域分析、系统稳定性判断类题型正确率≥85%，可独立写出每类常考题型的通用解题步骤
### 阶段 3：熟悉考研真题出题套路，提升解题速度与准确率，达到应试要求
- **行动项**：1. 完成近10年目标院校/统考拉氏变换相关真题训练，限时完成（每道题不超过3分钟）；2. 复盘所有错题，总结出题规律、常设陷阱；3. 对错题对应的知识点进行二次巩固，补齐剩余漏洞
- **推荐资源**：高频考点专项练习题；分层练习题(含答案与解析)（真题/难题部分）
- **检查点**：限时完成10道拉氏变换考研真题，正确率≥90%，平均解题时间≤3分钟，可清晰说出本模块近3年考研出题规律与高频考点

### 推送策略
- **日常推送规则**：周一至周五：每日推送1个核心考点卡片+2道对应专项练习题，占用时长约30分钟；周六：推送本周考点汇总复盘资料+1套15题的专项小测，占用时长约2小时；周日：推送本周错题解析+下周学习规划预告，占用时长约1小时
- **自适应规则**：若单类题型正确率低于70%，自动推送对应考点的讲解视频+5道补充练习题，强化巩固；每阶段检查点达标后自动解锁下一阶段学习内容，未达标则推送薄弱点专项补学包；自动归集个人错题，每3天推送1次错题复盘练习，巩固薄弱点
