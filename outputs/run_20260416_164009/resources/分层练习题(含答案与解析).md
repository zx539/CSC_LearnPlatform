**多重积分练习题及解析**

---

**题目1:** 计算下列定积分：
$$\int_{0}^{1} x^{2} \, dx$$

**答案与解析:**
$$\int_{0}^{1} x^{2} \, dx = \left[\frac{x^{3}}{3}\right]_{0}^{1} = \frac{1}{3}$$

**题目2:** 计算下列定积分：
$$\int_{-1}^{1} (x^{2} - x) \, dx$$

**答案与解析:**
$$\int_{-1}^{1} (x^{2} - x) \, dx = \left[x^{3} - \frac{x^{3}}{3}\right]_{-1}^{1} = \left(\frac{1}{3}\right) - \left(-\frac{1}{3}\right) = \frac{2}{3}$$

**题目3:** 计算下列定积分：
$$\int_{0}^{\pi} \sin(x) \, dx$$

**答案与解析:**
$$\int_{0}^{\pi} \sin(x) \, dx = -\cos(x)\bigg|_{0}^{\pi} = -\cos(\pi) + \cos(0) = 1$$

**题目4:** 计算下列定积分：
$$\int_{0}^{1} \sqrt{x} \, dx$$

**答案与解析:**
$$\int_{0}^{1} \sqrt{x} \, dx = \left[\frac{\sqrt{x}}{2}\right]_{0}^{1} = \frac{1}{2}$$

**题目5:** 计算下列定积分：
$$\int_{-1}^{1} \ln(x) \, dx$$

**答案与解析:**
$$\int_{-1}^{1} \ln(x) \, dx = \left[\ln(x)\right]_{-1}^{1} = \ln(1) - \ln(-1) = 0$$
