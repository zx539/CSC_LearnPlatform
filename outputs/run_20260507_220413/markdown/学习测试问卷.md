### Q2. 二重积分∬_D f(x,y)dσ的几何意义，以下表述正确的是？
- **题型**：single_choice
- **是否必填**：必填
- **评估维度**：概念理解
- **可选项**：
  - 当f(x,y)≥0时，积分结果是曲面z=f(x,y)与底面D围成的曲顶柱体的体积
  - 无论f(x,y)正负，积分结果都是曲面z=f(x,y)与底面D围成的曲顶柱体的体积
  - 积分结果始终等于积分区域D的面积
  - 积分结果始终等于被积函数f(x,y)在D上的最大值乘以D的面积
